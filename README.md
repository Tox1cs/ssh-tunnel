# Tox1c SSH-Tunnel

A professional, secure, and automated SSH Tunneling Manager.

## Features
* **Secure:** Users are jailed (No shell) and isolated in `tox1c-users` group.
* **Optimized:** Includes compiled UDP Gateway for Gaming/VoIP.
* **Tox1c-Dashboard:** Beautiful interactive CLI for management.
